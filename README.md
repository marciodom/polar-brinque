# Polar Brinque

 Aplicação Web desenvolvida como Progressive Web App (PWA), publicada via GitHub Pages, com suporte a funcionamento offline por meio de Service Worker.
O Polar Brinque é um sistema de gerenciamento de locação de brinquedos em praças públicas, criado para auxiliar no controle operacional e financeiro de eventos.

---

## 🚀 Demonstração

🔗 Acesse: https://marciodom.github.io/polar-brinque/

A aplicação pode ser instalada no dispositivo como aplicativo (PWA).
---

💡 Sobre a Aplicação
O sistema permite:
Configurar valores e tempo padrão de locação
Iniciar eventos informando:
Nome da criança
Tempo contratado
Brinquedo utilizado
Editar dados do evento após iniciado
Encerrar evento por:
Conclusão normal (tempo completo)
Desistência
Emissão de alerta sonoro (bipe) ao final do tempo
Geração de relatório completo para controle financeiro
A aplicação foi projetada para uso rápido, intuitivo e funcional em ambiente real de trabalho.
